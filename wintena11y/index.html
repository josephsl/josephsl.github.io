Welcome to Threshold Accessibility Response Center

In 2014, Microsoft announced two significant items: Windows 10 and Windows Insider Program (WIP). By the time Windows 10 Version 1507 (RTM/1024) shipped, more than five million people have signed up for a chance to test drive early builds of Windows 10.

As a screen reader developer and a Windows Insider, I (Joseph Lee) felt it would be a good idea to open a website dedicated to accessibility information on Windows 10, Universal Windows Platform and others. Along with information on new builds, this website will also house scripts and builds of screen readers that provide latest fixes for preview builds.

As of June 2016, the latest builds of Windows 10 are:

    Fast ring: 14361.0 (Redstone 1) Slow ring: 14342.1001 (Redstone 1) Release preview: 10586.420 (Threshold 2) Current Branch: 10586.420 (Threshold 2) Current Branch for Business: 10586.420 (Threshold 2) Long-Term Servicing Branch: 10240.16942 (Threshold 1)

    TARC goals

    The goals of TARC are:

    • Provide up-to-date accessibility information on Insider builds
    • Provide scripts for various screen readers and universal apps.
    • For NVDA users: provide try builds to fix issues encountered in Insider builds as soon as possible.
